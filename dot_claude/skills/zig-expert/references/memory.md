# Memory & Pointers\n\n## Slices vs Pointers\n- `[]T`: Slice (pointer + length).\n- `[*]T`: Many-pointer (no length, like C pointer).\n- `*T`: Single pointer.\n- `*const T`: Constant pointer.\n\n## Alignment\n- Use `align(64)` for cache-line alignment in critical structures.\n- `std.mem.Allocator` usually returns aligned memory.\n\n## Memory Safety\n- Use `defer` for simple cleanup.\n- Use `errdefer` for cleanup that only happens on error (e.g., in `init`).\n- `null` is not a value; use `?*T` for optional pointers.
