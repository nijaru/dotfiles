# SIMD in Zig\n\n## @Vector(N, T)\n- `N`: Number of elements (comptime).\n- `T`: Element type (e.g., `f32`, `i32`).\n\n## Operations\n- **Arithmetic:** `+`, `-`, `*`, `/` work element-wise.\n- **Reductions:** `@reduce(.Add, v)`, `@reduce(.Min, v)`, `@reduce(.Max, v)`.\n- **Select:** `@select(T, mask, a, b)`.\n\n## Memory Access\n- **Load:** `const v: @Vector(4, f32) = a[0..4].*`.\n- **Store:** `a[0..4].* = v`.\n\n## Example: L2 Distance\n```zig\nfn l2Squared(comptime dim: usize, a: []const f32, b: []const f32) f32 {\n    const V = @Vector(dim, f32);\n    const va: V = a[0..dim].*;\n    const vb: V = b[0..dim].*;\n    const diff = va - vb;\n    return @reduce(.Add, diff * diff);\n}\n```
